# steemgg - 基于区块链的HTML5游戏发布平台

## steemgg 即将正式发布

* https://steemgg.com
* 2018年7月下旬正式发布
* 网站所有代码均开源，欢迎参与贡献
* 基于 STEEM 区块链和 IPFS 分布式协议

## 帮助独立游戏成功

* 让优秀的独立游戏，不仅叫好，也叫座
* 让喝西北风成为历史

## 游戏好玩是重点

* 只要获得玩家的认可和赞同，玩家给游戏点赞，就能让游戏盈利

## 没有额外开发工作量

- 不需要添加任何额外的SDK

## 无需在游戏中刻意设置消费陷阱

* 玩家的认可和点赞说明了一切
* 让开发者更专注游戏玩法的迭代优化

## Game Jam 的作品也能盈利

* 不止是成品才能盈利
* Game Jam 的优秀 HTML5 作品同样可以得到玩家的认可和赞同
* 得到玩家的点赞就能盈利
